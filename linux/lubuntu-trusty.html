Lubuntu Trusty Setup

1 Installation

  1. Get the installer from lubuntu.net
  2. Make an installer out of the ISO image. Use unetbootin to make the USB installer
  3. On the target machine, the one you want to install Linux on, boot into the USB installer. Your BIOS or EFI may have specific keys to press during startup in order to activate the boot menu. On older hardware, you have to enter the BIOS and define the boot sequence. Most modern hardware allows you to press a key (usually Fn keys) to enter a boot menu
  4. For best results, make sure you are connected to the internet
  5. Follow the prompts, then reboot

2 Change the root password.

Yes there is sudo, and that is what you should really use, but change the root paswd anyway to something that you will actually remember. So that when you do need it, it will be on top of your mind and you don't have to wade through old notebooks. There will be times when you will need this. They will be emergency situations. sudo passwd root, then follow the prompts

3 Emacs setup.

sudo apt-get install emacs-nox emacs-goodies-el. emacs-nox is CLI one, it doesn't use X Windows, that's what nox stands for, no X.

4 Screen multiplexer.

Using the tabs on lxterminal is fine, but use a real multiplexer; sudo apt-get install tmux

5 Fix that prompt.

export the PS1 var, make it whatever you like, put this in ~/.bashrc 

export PS1=" _$ " # You know how to pwd anyway

6 Java SDK

The one from Oracle. You need to add an entry to apt repos, here's how to do it

$ sudo add-apt-repository ppa:webupd8team/java
$ sudo apt-get update
$ sudo apt-get install oracle-java8-installer

You cannot leave this, you need to accept the license agreement before it can commence installation. There are ways to auto accept the license, the steps are detailed in the Webupd8team website.

7 Vim.

This installed by default. You only need to setup your ~/.vimrc. Yes Vim is wonderful and it works out of the box, but the little tweaks you've been accustomed to needs to go to vimrc. Grab the vimrc here. Rename the downloaded file to to .vimrc and put it in your home folder. Before you do that, there are specific set of steps that you need to do before simply plunking that file into ~/.vimrc, better read the Vim Setup page

9 Email.

Use Mutt and OfflineIMap, it's Linux so use CLI all the way. OfflineImap makes a local copy of your IMAP folders, so it serves as a good backup when things go awry.

10 Fix the TMUX and Emacs colors

For some reason, tmux messes with the color theming of emacs. Out of laziness, just alias tmux so you can specify the TERM screen. In ~/.bashrc, add this 

alias tmux="TERM=xterm-256color-bce tmux"

Close up the editor, and source ~/.bashrc so the setting can take effect on the current terminal

Update: Instead of aliasing tmux, alias emacs instead, that way, emacs has its own term setting 

alias emacs="TERM=xterm-256color-bce emacs"
