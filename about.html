the site

It's a blog site. It has my notes on a couple of subjects. Most of the notes are technical in nature, I use them for teaching. It also contains notes that are non-technical, they mostly tilt towards soft-skills.

There is no explicit navigation on the front page because I haven't gotten to working on it yet; it is possible that I will never get to work on a clear, crisp and SEO friendly organization for this site, so here is how it is structured.

0. tedhagos.com

primary site, this part contains the non-technical notes.

1. android

This part of the site contains notes on Android programming. If you already know to code in Java and you are new to Android programming, these notes might be of some use to you.

2. html5

notes on HTML5 programming and other related technologies

3. misc

things that I cannot categorize, it is also possible that I don't write a lot of notes about the subject, hence it is on the misc bucket. An example of a note that will fall on this bucket is jekyll + github which contains my notes about setting up a site using jekyll and hosting it on github. I don't have a lot of notes about that subject, so it does not warrant a dedicated directory
