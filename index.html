Doc Convention

Things you need to type on a command or terminal window will appear in like this . So if you see something like mvn exec:java it means you need to type that exact command using a terminal window.

Things you need to type inside a file will appear like the one below. 

export tmux="TERM=screen-256color-bce tmux"

The formatting is in monospace and slightly indented. When you see a text like that, it is meant to be written on a file e.g config file or a program source listing. Another example of how this typestyle is used in this site is found below. 

// Hello.m     
#import<Foundation/Foundation.h>
#import<stdlib.h>

int main(int argc, const *char argv[]){
  NSLog(@"Hello Objective C\n");
  return EXIT_SUCCESS;
}
